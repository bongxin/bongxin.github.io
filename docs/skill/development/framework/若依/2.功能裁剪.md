# 功能裁减
主要是把原始系统中不需要的功能去掉，包括菜单、代码、按钮、功能入口等
## 去掉顶部菜单项
### 必删菜单项
去掉的菜单项包含`源码地址`,`文档地址`
### 可选菜单项
去掉的菜单项包含`搜索`,`布局大小`,`布局设置`
